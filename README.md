# image2ascii

An Image to ascii tool. Written in Rust.

# Example #
Original photo<br><br>
![](example/funny-cat.jpg)<br><br>
ASCII result<br><br>
<img src="example/funny-cat-ascii.png" width="100" height="100"/>
# Build #
Build via cargo
```
cargo build
```
# Usage #
```
./image2ascii IMAGE_PATH
```
